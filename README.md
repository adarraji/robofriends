# Robo Friends
My first React web application. Use the search box to find robo friend.

To run the application

1. Clone this repo
2. Run `npm install`
3. Run `npm start`# robofriends

## Built With

* HTML
* CSS
* Javascript
* React

## Deployment
Deployed on Github Pages

You can see the website [here](https://adarraji.github.io/robofriends/)
